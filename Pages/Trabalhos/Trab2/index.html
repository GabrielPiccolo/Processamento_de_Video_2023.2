TRABALHO - PROCESSAMENTO DE VÍDEO 2023.2

Parte 2: Modelagem Funcional do Sistema de Processamento Visual (MFS):


TEMA: Analise condicional do uso de EPI's por identificação de cores da roupa.


INÍCIO

Essa é a etapa inicial do processo. Representa o ponto de partida do sistema ou programa.

Código de abertura do executável

Nessa etapa, o código responsável pela abertura do executável é executado. Essa parte do processo pode incluir inicialização de variáveis, configurações iniciais e preparação do ambiente.

Opções

Nessa etapa, o sistema oferece opções ao usuário para escolher o próximo passo a ser executado. Essas opções podem estar relacionadas a diferentes funcionalidades do sistema.

ENCERRAR O PROGRAMA

Se o usuário escolher essa opção, o programa é encerrado e o processo termina.

Abrir Câmera

Se o usuário selecionar essa opção, o programa é projetado para abrir a câmera e preparar-se para realizar a captura de imagens.

Código de abertura da câmera

Essa etapa envolve o código necessário para abrir a câmera. Pode incluir o acesso ao dispositivo de câmera e a inicialização dos recursos necessários para a captura de imagens.

Código de varredura de movimentos (WHILE)

Essa etapa envolve um loop (WHILE) que verifica se há algum movimento detectado pela câmera. O programa continua a executar essa etapa até que não haja mais movimentos detectados.

MENSAGEM DE ERRO NA ABERTURA DA CÂMERA

Se ocorrer algum erro durante a abertura da câmera, essa etapa é acionada para exibir uma mensagem de erro na tela.

Erro

Essa etapa representa um estado de erro ou falha no processo.

Código para verificação de indivíduo

Essa etapa envolve o código necessário para verificar a presença de um indivíduo na imagem capturada pela câmera. Pode incluir algoritmos de detecção de objetos.

Código para verificação da cor da camisa dessa pessoa

Essa etapa envolve o código necessário para verificar a cor da camisa da pessoa detectada na imagem capturada. Pode incluir algoritmos de processamento de imagem para identificar a cor da roupa.

Código de Mostrar mensagem na tela

Essa etapa envolve o código para exibir uma mensagem na tela do sistema ou programa.

MENSAGEM DE COR INDEVIDA

Se a cor da camisa da pessoa for incorreta, essa etapa é acionada para exibir uma mensagem de cor incorreta na tela.

Código de verificação dos EPI’s daquela cor

Essa etapa envolve o código necessário para verificar se os Equipamentos de Proteção Individual (EPIs) na cor correspondente estão sendo usados pela pessoa.

Código de verificação da presença de EPI’s

Essa etapa envolve o código necessário para verificar se os EPIs estão presentes na imagem capturada pela câmera. Pode incluir algoritmos de detecção de objetos para identificar a presença dos EPIs.

MENSAGEM DE ACESSO LIBERADO

Se os EPIs adequados estiverem sendo usados, essa etapa é acionada para exibir uma mensagem de acesso liberado na tela.

MENSAGEM DE ACESSO NEGADO

Se os EPIs estiverem incorretos ou ausentes, essa etapa é acionada para exibir uma mensagem de acesso negado na tela.

Código de tirar foto da pessoa

Essa etapa envolve o código necessário para capturar uma foto da pessoa detectada pela câmera.
